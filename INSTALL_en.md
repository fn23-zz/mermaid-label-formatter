# Installation Guide (VSIX)

This guide explains how to install the extension using the provided `.vsix` file.

---

## 1. Preparation

Before starting, please ensure you have the following file:

* **`mermaid-label-formatter-0.0.1.vsix`**

---

## 2. Installation Steps

Please follow these steps to install the extension in Visual Studio Code:

1. **Open Visual Studio Code**
2. **Open the Extensions View**
    * Click the **Extensions icon** in the Activity Bar on the left side of the window (or press `Ctrl+Shift+X`).
3. **Open the Menu**
    * Click the **"..." (More Actions)** icon at the top of the Extensions pane.
4. **Select Install from VSIX**
    * Select **"Install from VSIX..."** from the dropdown menu.
5. **Choose the File**
    * A file browser will appear. Select the `mermaid-label-formatter-0.0.1.vsix` file you downloaded and click **"Install"**.

---

## 3. Verification

After installation is complete:

* **Restart VS Code**
    * It is recommended to restart VS Code to ensure the extension is fully loaded.
* **Usage**
    * Open any `.md` (Markdown) file. The extension will automatically format Mermaid labels to prevent rendering errors.
