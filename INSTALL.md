# インストールガイド (VSIX版)

このガイドでは、提供された `.vsix` ファイルを使用して拡張機能をインストールする方法を説明します。

---

## 1. 準備するもの

開始する前に、以下のファイルが手元にあることを確認してください。

* **`mermaid-label-formatter-0.0.1.vsix`**

---

## 2. インストール手順

以下の手順に従って、Visual Studio Code に拡張機能をインストールしてください。

1. **Visual Studio Code を起動します**
2. **拡張機能ビューを開きます**
    * 左側のサイドバーにある **拡張機能アイコン** をクリックします（または `Ctrl+Shift+X` を押します）。
3. **メニューを開きます**
    * 拡張機能パネルの上部にある **「...」（その他のアクション）** アイコンをクリックします。
4. **「VSIX からインストール」を選択します**
    * ドロップダウンメニューから **「VSIX からインストール...」** を選択します。
5. **ファイルを選択します**
    * ファイル選択画面が表示されます。ダウンロードした `mermaid-label-formatter-0.0.1.vsix` ファイルを選択し、**「インストール」** をクリックします。

---

## 3. インストールの確認

インストールが完了したら、以下の点を確認してください。

* **VS Code の再起動**
    * 拡張機能が完全に読み込まれるよう、VS Code の再起動をお勧めします。
* **使い方**
    * `.md` (Markdown) ファイルを開いてください。Mermaidのラベルが自動的に整形され、表示エラーを防ぎます。
