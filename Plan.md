# Mermaid Label Formatterの目的
Markdown PDF VS Code の Markdown PDF:export(pdf)機能で、mermaidの描写が失敗することが多くこれを回避する拡張機能を開発する

## 現象の解析
"・" あるいは "？" という文字を使っているとエラーになる。

## 回避策
ダブルクォーテーション `""` が最強の回避策である理由__: Mermaid の構文において、ラベルを `"` で囲むことは、その中身を「構文記号」ではなく「単なる文字列（リテラル）」として扱うようパーサーに指示することになります。 これにより、パーサーが全角記号を読み取った際に発生する構文エラーを根本から防ぐことができるため、記号の種類を問わず最も安定して動作します。


## 実装方針
VS Code 拡張機能として実装する。
Markdown ファイル内の Mermaid コードブロックを検出し、その中のノード定義 `A[テキスト]` を `A["テキスト"]` に書き換える。
